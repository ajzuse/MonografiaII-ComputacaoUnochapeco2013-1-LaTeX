\resumo{Este trabalho apresenta a importância dos dispositivos móveis para a 
sociedade atual e a utilização destes dispositivos no meio acadêmico na Unochapecó, 
demonstrando por meio de questionário a sua utilização pelos acadêmicos. Também por 
meio da mesma pesquisa levanta a importância dos itens presentes no Sistema acadêmico 
Minha Uno nos mais diversos perfis. A partir dos dados coletados pelo questionário 
são selecionados os itens mais relevantes do perfil acadêmico e a partir destes é 
demonstrada a implementação de um servidor de extração de dados do sistema acadêmico 
Minha Uno e também a implementação de uma aplicação móvel para acesso as informações 
extraídas pelo servidor. Com uma abordagem prática e sem o acesso direto as informações 
do sistema acadêmico atual, é utilizada no servidor uma biblioteca de extração de dados 
a partir de código HTML, e posteriormente as informações extraídas são disponibilizadas 
em um servidor REST consumido pelo aplicativo móvel desenvolvido em JavaScript 
utilizando-se do Titanium SDK. Ao final do trabalho são sugeridas possibilidades de 
continuidade do mesmo com o objetivo de, até mesmo, tornar possível a utilização do 
projeto por outras instituições.}
{Aplicativo Móvel; Extração de Informações; Sistema Acadêmico; Unochapecó}

% \abstract {Abstract content} {Keywords}
\abstract{This paper presents the importance of mobile devices for today's society and the use of these devices by academics in Unochapecó, demonstrating through a questionnaire to be used by scholars. Also by the same research raises the importance of the items present in the academic system \emph{Minha Uno} in various profiles. From the data collected by the questionnaire are selected the most relevant items of the academic profile and from these is shown the implementation of a server data extraction from academic system \emph{Minha Uno} and also the implementation of a mobile application to access the information from the server . With a practical approach and without direct access to the information of the current academic system, is used in the server library for data extraction from HTML, and then the extracted information is made available on a server REST consumed by mobile app developed in JavaScript using the Titanium SDK. At the end of the work are suggested possibilities of continuing the same in order to even make it possible to use the project by other institutions.}
{Mobility; Server; Extraction; Communication; Unochapecó}
